# 电路原理图连线表

> 在嘉立创EDA中画原理图时，按此表连线。
> 格式：`元件-引脚` 连接到 `元件-引脚`，注明中间经过的元件。

## M1: NE555 PWM发生器

### 元件清单
| 编号 | 元件 | 值 | 封装 |
|------|------|----|------|
| U1 | NE555 | - | DIP-8 |
| W1 | 电位器 | 20kΩ | 直插 |
| W2 | 电位器 | 47kΩ | 直插 |
| D1 | 1N4148 | - | DO-35 |
| D2 | 1N4148 | - | DO-35 |
| C1 | 电容 | 10nF | 瓷片 |
| C2 | 电容 | 10nF | 瓷片 |

### 连线
```
VCC ──┬── NE555.pin8(VCC)
      │
      ├── NE555.pin4(RST)
      │
      └── W1.pin1 ── W1.wiper ── NE555.pin7(DIS)
                                   │
                    D1.阳极 ───────┤
                    D1.阴极 ──┬── W2.wiper
                              │
                    D2.阴极 ──┤
                    D2.阳极 ──┘
                               │
                    W2.pin1 ──┬── NE555.pin6(THR)
                    W2.pin2 ──┘── NE555.pin2(TR)
                                   │
                              C1 ──┤
                                   │
                                  GND

NE555.pin3(OUT) ──→ PWM_OUT → M2

NE555.pin5(CTRL) ── C2 ── GND

NE555.pin1(GND) ── GND
```

### 工作原理
- W1 调节充放电频率（3-5kHz）
- D1+D2 二极管引导：充电经D1，放电经D2
- W2 调节占空比（30%-70%）
- C1 为定时电容，与R_freq配合决定频率
- C2 旁路电容，滤除控制电压噪声

## M2: 功率放大级

### 元件清单
| 编号 | 元件 | 值 | 封装 |
|------|------|----|------|
| Q1 | IRL540N | N-MOSFET | TO-220 |
| R1 | 电阻 | 100Ω | 直插 |
| R2 | 电阻 | 10kΩ | 直插 |
| D1 | 1N4007 | 续流二极管 | DO-41 |
| M1 | 直流电机 | 12V | - |

### 连线
```
PWM_IN ── R1(100Ω) ──┬── Q1.栅极(G)
                      │
                     R2(10kΩ)
                      │
                     GND

VCC(+12V) ── M1(电机+) ──┬── Q1.漏极(D)
                          │
                         D1.阴极 ←── D1.阳极 ── Q1.漏极(D)

Q1.源极(S) ── GND
```

### 工作原理
- IRL540N 为逻辑电平MOSFET，5V栅极驱动即可完全导通
- R1 限制栅极充电电流，防止振铃
- R2 下拉电阻，无信号时确保MOSFET关断
- D1 续流二极管，保护MOSFET免受电机反向EMF

## M3: 转速检测

### 元件清单
| 编号 | 元件 | 值 | 封装 |
|------|------|----|------|
| U1 | 光电传感器 | H21A1 | DIP-4 |
| U2 | 74HC14 | 施密特触发器 | DIP-14 |
| R3 | 电阻 | 10kΩ | 直插 |
| R4 | 电阻 | 100kΩ | 直插 |

### 连线
```
VCC ── R3(10kΩ) ──┬── U1.集电极
                   │
              U1.发射极
                   │
                  GND

U1.集电极 ──┬── U2.1A(输入)
            │
           R4(100kΩ)
            │
           GND

VCC ── U2.14(VCC)
U2.7(GND) ── GND

U2.1Y(输出) ──→ F_OUT → M5(CD4060计数)
```

### 工作原理
- 光电传感器检测电机转盘缝隙，产生脉冲信号
- R3 上拉电阻，保证无遮光时输出高电平
- 74HC14 施密特触发器整型，消除抖动产生干净方波
- R4 提供滞后电压，增强抗干扰能力

## M5: 时基与定时控制

### 元件清单
| 编号 | 元件 | 值 | 封装 |
|------|------|----|------|
| U1 | CD4060 | 14位二进制计数器 | DIP-16 |
| U2 | 74HC74 | D触发器 | DIP-14 |
| U3 | 74HC121 | 单稳态触发器 | DIP-14 |
| Y1 | 晶振 | 32.768kHz | HC-49S |
| R5 | 电阻 | 22MΩ | 直插 |
| R6 | 电阻 | 10MΩ | 直插 |
| R7 | 电阻 | 100kΩ | 直插 |
| R8 | 电阻 | 47kΩ | 直插 |
| R9 | 电阻 | 47kΩ | 直插 |
| C3 | 电容 | 22pF | 瓷片 |
| C4 | 电容 | 22pF | 瓷片 |
| C5 | 电容 | 10µF | 电解 |
| C6 | 电容 | 1µF | 电解 |
| C7 | 电容 | 1µF | 电解 |

### 连线
```
Y1(晶振) ──┬── CD4060.OSC_IN(pin11)
            └── CD4060.OSC_OUT(pin10)

R5(22MΩ) ── OSC_IN ↔ OSC_OUT（跨接反馈电阻）
R6(10MΩ) ── OSC_IN ↔ OSC_OUT（偏置电阻）
C3(22pF) ── OSC_IN ── GND
C4(22pF) ── OSC_OUT ── GND

VCC ── R7(100kΩ) ──┬── CD4060.RST(pin12)
                    │
                   C5(10µF)
                    │
                   GND

CD4060.Q14(pin3) ── 74HC74.CLK(pin3)

74HC74.D(pin2) ── 74HC74./Q(pin6)（自触发÷2）
74HC74.PR(pin1) ── VCC
74HC74.CLR(pin13) ── VCC

74HC74.Q(pin5) ── 1Hz时钟

1Hz ──┬── 74HC121.A1(pin3)  ──→ LATCH脉冲（τ=47kΩ×1µF≈47ms）
      └── 74HC121./Q(pin1) ──→ 74HC121(B).A1 ──→ CLEAR脉冲（τ=47kΩ×1µF≈47ms）

VCC ── 74HC121.VCC(pin14)
74HC121.GND(pin7) ── GND
```

### 工作原理
- CD4060 + 32.768kHz晶振：内部振荡+14级分频，Q14输出2Hz
- 74HC74 D触发器接成÷2模式，2Hz→1Hz精准秒脉冲
- 74HC121 单稳态触发器：
  - 第一个：1Hz上升沿触发，产生LATCH锁存脉冲
  - 第二个：LATCH下降沿触发，延迟后产生CLEAR清零脉冲
  - 时间常数 τ = R_ext × C_ext = 47kΩ × 1µF = 47ms

## M6: 计数与显示

### 元件清单
| 编号 | 元件 | 值 | 封装 |
|------|------|----|------|
| U1 | CD4518 | 双BCD计数器 | DIP-16 |
| U2 | CD4511 | BCD-7段译码器 | DIP-16 |
| U3 | CD4511 | BCD-7段译码器 | DIP-16 |
| DS1 | 共阴数码管 | 0.56寸 | - |
| DS2 | 共阴数码管 | 0.56寸 | - |
| R1-R7 | 电阻×7 | 220Ω | 直插 |
| R8-R14 | 电阻×7 | 220Ω | 直插 |

### 连线
```
F_IN(转速脉冲) ── CD4518.1CLK(pin1)
LATCH ── CD4518.1EN(pin2)
CLEAR ── CD4518.1RST(pin7)

CD4518.1Qa(pin3) ── U2.A(pin3)
CD4518.1Qb(pin4) ── U2.B(pin4)
CD4518.1Qc(pin5) ── U2.C(pin5)
CD4518.1Qd(pin6) ── U2.D(pin6)

CD4518.1Qd(pin6) ── CD4518.2CLK(pin9)（级联：个位十进制→十位计数）

CD4518.2Qa(pin11) ── U3.A(pin3)
CD4518.2Qb(pin12) ── U3.B(pin4)

U2/U3:
  LT(pin5) ── VCC（灯测试，常态无效）
  BI(pin4) ── VCC（消隐，常态无效）
  LE(pin6) ── GND（锁存使能，常态直通）
  VCC(pin16) ── VCC
  GND(pin8) ── GND

U2.a-g(pin13,12,11,10,9,15,14) ── R(220Ω)×7 ── DS1.a-g
U3.a-g ── R(220Ω)×7 ── DS2.a-g

DS1/DS2.公共端 ── GND（共阴接法）
```

### 工作原理
- CD4518 双十进制BCD计数器：
  - 第一组计个位（0-9），LATCH上升沿锁存当前值
  - 第二组计十位（0-9），由个位Qd进位触发
- CD4511 BCD转7段译码器，直接驱动共阴数码管
- 220Ω限流电阻保护LED段（(5V-2V)/220Ω≈13.6mA/段）
- CLEAR脉冲在LATCH后延迟清除计数器，准备下一秒计数

## 电源模块

### 元件清单
| 编号 | 元件 | 值 | 封装 |
|------|------|----|------|
| U4 | LM7805 | 5V稳压器 | TO-220 |
| C8 | 电容 | 100µF | 电解 |
| C9 | 电容 | 100nF | 瓷片 |

### 连线
```
12V电源 ── C8(100µF) ──┬── LM7805.IN(pin1)
                        │
                       GND

LM7805.OUT(pin3) ──┬── C9(100nF) ── GND
                    │
                    └── VCC(+5V) → 所有芯片

LM7805.GND(pin2) ── GND
```
