# BOM 器件清单（混合方案）

> **最后更新：2026.6.21** — 基于 Proteus 仿真验证结果

## 方案架构

- **调速部分：** NE555 + 二极管导引 + PC817A 光耦隔离 + IRL540 MOSFET

## 九、焊接注意事项

### 必须正确连接的引脚

| 芯片 | 引脚 | 连接 | 悬空后果 |
|------|------|------|----------|
| NE555 | RST（脚4） | → +12V | 不振荡 |
| NE555 | GND（脚1） | → GND | 不工作 |
| NE555 | CV（脚5） | → C2(10nF) → GND | 噪声干扰 |
| CD40110 | TE（脚4） | → GND | 不计数 |
| CD40110 | CPD（脚7） | → GND | 不稳定 |
| CD40106 | 未用输入 | → GND 或 5V | 功耗增大 |

### 光耦隔离注意
- PC817A 的 **LED侧**（控制端）和 **光敏管侧**（驱动端）**不能共地**
- 两侧各自有独立的电源和地回路
- PCB布局时两侧保持间距

### MOSFET 驱动注意
- IRL540 栅极上拉电阻必须用 **470Ω**，不能用 10kΩ
- 电机两端并联续流二极管（1N4007），阴极接 VCC
