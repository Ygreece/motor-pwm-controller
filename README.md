# 直流电机转速检测与脉宽调速器

电子设计综合实验 — 直流电机的转速检测与脉宽调速器的设计

## 项目概述

- **项目类型：** 电子设计综合实验（单人项目）

## 设计要求

1. **PWM发生器** — 多谐振荡器，频率3~5kHz，占空比30%~70%
2. **功率放大器** — 电机额定电压≤12V，额定电流≤0.5A
3. **转速检测电路** — 光电转换方法，输出脉冲信号
4. **转速显示电路** — 2位数码管显示 (r/s)

## 系统方案

采用**混合架构**：
- **调速部分：** NE555 + 二极管导引 PWM → PC817A 光耦隔离 → IRL540 MOSFET 驱动电机
- **测速显示部分：** PC817 电刷噪声检测 → CD40106 整形/时基 → CD40110 计数/译码/驱动 → 数码管

```
┌─────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────┐
│  NE555 PWM  │    │  PC817A 光耦  │    │  IRL540      │    │  直流电机  │
│  发生器      │───→│  隔离驱动     │───→│  MOSFET 功率  │───→│          │
│ (频率/占空比 │    │              │    │  放大器       │    │          │
│  可调)       │    │              │    │              │    │          │
└─────────────┘    └──────────────┘    └──────────────┘    └────┬─────┘
                                                                │ 电刷噪声
                                                                ▼
┌─────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────┐
│  2位数码管   │←───│  CD40110     │←───│  CD40106     │←───│  PC817   │
│  转速显示    │    │  计数/译码/   │    │  整形/时基    │    │  转速检测  │
└─────────────┘    └──────────────┘    └──────────────┘    └──────────┘
```
