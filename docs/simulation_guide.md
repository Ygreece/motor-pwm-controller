# Proteus 仿真指南

> **最后更新：2026.6.21** — 从 Multisim 切换到 Proteus

## 一、仿真环境

- **软件：** Proteus 8 Professional
- **NE555 模式：** Analog（右键属性确认）
- **仿真设置：** System → Set Animation Options → 默认 SPICE 参数

---

## 二、PWM 发生器仿真

### 2.1 电路搭建

1. 放置 NE555 元件（搜索 NE555）
2. 右键属性 → 确认 Model Type 为 **Analog**
3. 按 design.md 模块1 接线：
   - Pin8(VCC) → +12V
   - Pin1(GND) → GND
   - Pin4(RST) → +12V
   - Pin5(CV) → C2(10nF) → GND
   - Pin6(THR) + Pin2(TRG) → 节点A
   - Pin7(DIS) → R3(100kΩ) → 节点A
   - 节点A → C1(100nF) → GND
   - 节点A → D(1N4148，正极朝节点A) → R2(100kΩ) → +12V
   - Pin3(OUT) → 后级电路

### 2.2 示波器测试

1. 放置示波器（Virtual Instrument Mode → OSCILLOSCOPE）
2. Channel A 接 NE555 Pin3(OUT)
3. 运行仿真
4. 预期：方波输出，调节 R2/R3 可改变频率/占空比

### 2.3 常见问题

| 现象 | 原因 | 解决 |
|------|------|------|
| 输出锯齿波 | Pin7 充放电不对称 | 检查 D(1N4148) 方向 |
| 仿真不收敛 | NE555 模式错误 | 确认 Analog 模式 |
| 无输出 | Pin4(RST) 悬空 | 接 +12V |

## 三、功率驱动仿真

### 3.1 PC817A 光耦接线

```
NE555 Pin3 ── 330Ω ── PC817A Pin1
               PC817A Pin2 ── GND
               PC817A Pin4 ──┬── 470Ω ── +12V
                             └── IRL540 Gate
               PC817A Pin3 ── GND
```

### 3.2 MOSFET + 电机接线

```
+12V ── 电机 ── IRL540 Drain
                 IRL540 Source ── GND
```

### 3.3 关键参数

| 元件 | 推荐值 | 说明 |
|------|--------|------|
| R_LED | 330Ω | LED 限流 |
| R_pull | **470Ω** | 栅极上拉（不能用 10kΩ！） |

### 3.4 波形验证

用示波器看 IRL540 Gate 波形：
- 470Ω 上拉 → 接近方波 ✅
- 10kΩ 上拉 → 指数曲线 ❌

## 四、转速检测仿真

### 4.1 实际硬件 vs 仿真方案

| 项目 | 实际硬件 | Proteus 仿真 |
|------|---------|-------------|
| 检测方式 | 槽型光电传感器 + 遮光码盘 | PC817 电刷噪声法 |
| 信号质量 | 干净方波 | 需要 CD40106 整形 |
| 原理 | 光电门遮光产生脉冲 | 电刷换向噪声脉冲 |

### 4.2 信号源替代（推荐）

Proteus 中最简单的验证方式，用信号源替代转速脉冲：
- **频率：** 7~10 Hz（对应电机转速）
- **幅度：** 9V 方波
- **连接：** 信号源 → CD40106 输入

### 4.3 CD40106 时基振荡

```
CD40106 门D 输入 ──┬── W2(50kΩ) + R29(25kΩ) ── CD40106 门D 输出
                   └── C1(22μF) ── GND
```

调节 W2 至 **47%**，使振荡周期 ≈ 1 秒。

### 4.4 LE/RST 时序

```
CD40106 门F 输出(Pin8) → 所有 CD40110 Pin4 (LE)
CD40106 门C 输出(Pin6) → 所有 CD40110 Pin5 (RST)
```

时序：先 LE（锁存）→ 后 RST（清零）

## 五、混合仿真注意事项

### 5.1 收敛问题

NE555（Analog）与 CD40106/CD40110（Digital）放在同一原理图会报收敛错误。

**解决方案：**
1. **分开仿真**（推荐）：PWM 驱动和转速显示分别放在不同 .pdsprj 文件
2. **Exclude from simulation**：右键元件 → 从仿真中排除
3. **调整收敛参数**：
   ```
   System → Set Animation Options → SPICE Options
   RELTOL = 0.01
   ITL4 = 200
   GMIN = 1e-10
   ```

### 5.2 元件模式确认

| 元件 | 应设为 | 检查方法 |
|------|--------|---------|
| NE555 | Analog | 右键 → Properties → Model Type |
| 电阻 | Analog | 右键 → Properties |
| CD40106 | Digital | 默认 |
| CD40110 | Digital | 默认 |
