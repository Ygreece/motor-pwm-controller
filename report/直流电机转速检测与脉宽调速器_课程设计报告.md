## 一、设计任务与要求

### 1.1 设计任务

设计并制作一个直流电机转速检测与脉宽调速器，实现以下功能：

1. **PWM 脉宽调速** — 通过调节 PWM 信号的占空比控制直流电机转速
2. **转速检测** — 实时检测电机转速并转换为数字信号
3. **转速显示** — 通过数码管显示电机转速（单位：r/s）

### 1.2 技术指标

| 参数 | 要求 |
|------|------|
| PWM 频率 | 3 ~ 5 kHz |
| 占空比范围 | 30% ~ 70% |
| 电机额定电压 | ≤ 12V |
| 电机额定电流 | ≤ 0.5A |
| 转速显示 | 2 位数码管 |
| 隔离要求 | 光耦隔离（PWM 驱动与控制电路之间） |

### 1.3 设计条件

- 供电电压：+12V
- 仿真工具：Proteus 8
- 电机类型：直流有刷电机

### 2.2 模块划分

系统分为四大功能模块：

| 模块 | 功能 | 核心器件 |
|------|------|---------|
| **PWM 发生器** | 产生频率和占空比可调的方波 | NE555 + 1N4148 |

## 三、各模块电路设计

### 3.1 PWM 发生器电路

#### 3.1.1 电路原理

采用 NE555 定时器构成多谐振荡器，通过二极管引导充放电路径，实现频率和占空比独立可调。

**充电路径**（Pin3=HIGH，Pin7 开路）：
```
+12V → R2(100kΩ) → D(1N4148，正向导通) → C1(100nF) → GND
```

**放电路径**（Pin3=LOW，Pin7 接地）：
```
C1(100nF) → R3(100kΩ) → Pin7(内部接地) → GND
            D(1N4148，反向截止) 隔断 R2
```

#### 3.1.2 电路图

```
                    +12V
                     │
                     R2 (100kΩ 电位器 - 频率调节)
                     │
                     ├──── D (1N4148，正极朝右)
                     │         │
                     │     ┌───┴───┐
                     │     │  VCC  │
                     │     │       │
                     │     │  RST  ├── +12V
                     │     │       │
                     │     │  OUT  ├── 330Ω → PC817A Pin1
                     │     │       │
                     │     │  CV   ├── C2(10nF) → GND
                     │     │       │
                     │     │  THR  ├──┐
                     │     │       │  │
                     │     │  TRG  ├──┤
                     │     │       │  │
                     │     │  DIS  ├──┤── R3 (100kΩ 电位器 - 占空比调节)
                     │     │       │  │
                     │     │  GND  │  │
                     │     └───┬───┘  │
                     │         │      │
                     │        GND     │
                     │               │
                     │          C1 (100nF)
                     │               │
                     │              GND
```

#### 3.1.3 参数计算

频率公式：
$$f = \frac{1.44}{(R_2 + R_3) \times C_1}$$

占空比公式：
$$D = \frac{R_2}{R_2 + R_3} \times 100\%$$

当 R2 = R3 = 100kΩ，C1 = 100nF 时：
- 频率：f = 1.44 / (200kΩ × 100nF) = **72 Hz**（中点值）
- 占空比：D = 100kΩ / 200kΩ = **50%**（中点值）

调节范围：
- 频率：约 72 Hz ~ 14.4 kHz（取决于电位器位置）
- 占空比：约 0% ~ 100%

> 注：实际 PWM 频率要求 3~5kHz，可通过调节电位器达到。

#### 3.1.4 元件清单

| 元件 | 型号/值 | 作用 |
|------|---------|------|
| U1 | NE555 | PWM 信号发生 |
| R2 | 100kΩ 电位器 | 频率调节 |
| R3 | 100kΩ 电位器 | 占空比调节 |
| C1 | 100nF (104) | 定时电容 |
| C2 | 10nF | Pin5 去耦滤波 |
| D | 1N4148 | 隔离充放电路径 |
| R_limit | 330Ω | PC817A LED 限流 |

### 3.2 功率驱动电路（光耦隔离 + MOSFET）

#### 3.2.1 设计要求

题目要求 PWM 驱动与控制电路之间必须光耦隔离，防止电机反电动势干扰控制电路。

#### 3.2.2 电路设计

```
NE555 Pin3 ── 330Ω ── PC817A Pin1 (LED 阳极)
                       PC817A Pin2 (LED 阴极) ── GND
                       PC817A Pin4 (光敏管集电极) ──┬── 470Ω ── +12V
                                                   │
                                                   └── IRL540 Gate
                       PC817A Pin3 (光敏管发射极) ── GND

+12V ── 电机 ── IRL540 Drain
                 IRL540 Source ── GND
```

#### 3.2.3 工作原理

| NE555 输出 | PC817A LED | Pin4 集电极 | IRL540 Gate | MOSFET | 电机 |
|------------|-----------|------------|------------|--------|------|
| HIGH (12V) | 亮 | 拉低 (≈0V) | 0V | 关断 | 停 |
| LOW (0V) | 灭 | 上拉 (12V) | 12V | 导通 | 转 |

> **注意：** PC817A 光耦具有反相特性，NE555 输出高电平时 MOSFET 关断，低电平时导通。

#### 3.2.4 关键参数

**上拉电阻选择（470Ω）：**

上拉电阻与 MOSFET 栅极电容形成 RC 时间常数：
- MOSFET 栅极电容 C_iss ≈ 1600pF（IRL540）
- R = 10kΩ 时：τ = 10kΩ × 1600pF = **16μs**（上升沿为指数曲线，太慢）
- R = 470Ω 时：τ = 470Ω × 1600pF = **0.75μs**（上升沿接近方波）

经仿真验证，**470Ω 上拉电阻**可以保证 MOSFET 快速开关。

**LED 限流电阻（330Ω）：**
- LED 正向压降 ≈ 1.2V
- I_LED = (12V - 1.2V) / 330Ω ≈ **32.7mA**
- PC817A 最大允许电流 50mA，安全范围内

#### 3.2.5 元件清单

| 元件 | 型号/值 | 作用 |
|------|---------|------|
| U2 | PC817A | 光耦隔离 |
| R_pull | 470Ω | 栅极上拉 |
| R_LED | 330Ω | LED 限流 |
| Q1 | IRL540 | N-MOSFET 功率开关 |
| M1 | 直流电机 | 负载 |

### 3.3 转速检测电路

#### 3.3.1 检测原理

采用**电刷噪声法**：直流有刷电机在运转时，电刷与换向器之间的接触会产生高频噪声脉冲，每个脉冲对应电机旋转一定角度。通过光耦采集这些噪声脉冲，即可测量转速。

> 优点：无需在电机上安装遮光盘或磁铁，利用电机自身特性检测。

#### 3.3.2 电路设计

```
电机电刷噪声 ── PC817 (检测光耦) ── CD40106 (施密特整形) ── 数字脉冲
                                       │
                                  偏置/灵敏度调节
                                  W2(1000kΩ) + R29 + R35
```

#### 3.3.3 波形整形

CD40106 六施密特反相器用于将 PC817 输出的缓慢模拟信号整形为干净的数字方波：
- 施密特触发器具有迟滞特性，可消除信号抖动
- 输出标准 CMOS 电平，供后级 CD40110 计数

---

### 3.4 转速显示电路

#### 3.4.1 电路组成

```
CD40106 输出脉冲 ── CD40110 (U1) ──→ CD40110 (U2) ──→ 共阴极数码管
                     计数/译码/驱动    计数/译码/驱动     (2位显示)
                         ↑                  ↑
                    差分时序电路          CARRY 进位
                    LE(锁存) + RST(清零)
```

#### 3.4.2 CD40110 功能

CD40110 是集计数、译码、驱动于一体的芯片：

| 引脚 | 功能 | 说明 |
|------|------|------|
| Pin9 (CLKUP) | 加计数时钟 | 上升沿触发 |
| Pin10 (CARRY) | 进位输出 | 计数满 10 输出脉冲 |
| Pin4 (LE) | 锁存使能 | HIGH 时锁存显示数据 |
| Pin5 (RST) | 复位 | HIGH 时清零计数 |
| Pin6 (TE) | 计数使能 | LOW 时允许计数 |

#### 3.4.3 级联方式

4 片 CD40110 级联（实际使用 2 位显示）：
```
U1 Pin10 (CARRY) → U2 Pin9 (CLKUP)
U2 Pin10 (CARRY) → U3 Pin9 (CLKUP)  [如需扩展]
所有 Pin6 (TE) → GND（始终允许计数）
```

#### 3.4.4 时序控制

采用差分电路产生 LE（锁存）和 RST（清零）信号：
- **时序要求：先 LE（锁存当前计数）→ 后 RST（清零重新计数）**
- CD40106 振荡器产生 1 秒周期时基
- 差分电路（C3/C4 + R32）将方波边沿转换为短脉冲

| 测试项目 | 预期 | 实测 |
|---------|------|------|
| 输出波形 | 方波 | ✅ 方波 |

#### 4.2.2 调节特性

- **R2（频率旋钮）**：顺时针增大阻值 → 频率降低
- **R3（占空比旋钮）**：顺时针增大阻值 → 占空比减小（因光耦反相，实际电机转速增大）

### 4.3 功率驱动仿真

#### 4.3.1 MOSFET 栅极波形

| 上拉电阻 | 波形质量 | MOSFET 开关状态 |
|---------|---------|----------------|
| 10kΩ | 指数上升沿，幅度不足 | 半开不开，电机不转 |
| **470Ω** | **接近方波，幅度充足** | **完全开关，电机正常** |

#### 4.3.2 电机转速控制

| 占空比 (NE555) | MOSFET 导通比 | 电机状态 |
|---------------|--------------|---------|
| 30% | 70%（反相） | 快速 |
| 50% | 50% | 中速 |
| 70% | 30%（反相） | 慢速 |

### 4.4 转速检测仿真

使用信号源（7~10Hz，9V 方波）替代 PC817 检测信号：
- CD40106 整形输出干净方波 ✓
- CD40110 计数正常 ✓
- LE/RST 时序正确（先锁存后清零）✓
- 数码管显示正常 ✓

## 五、关键问题与解决方案

### 5.1 Proteus 混合仿真收敛问题

**问题：** NE555（模拟器件）与 CD40106/CD40110（数字器件）放在同一原理图时，即使没有物理连接，也会报 "Too many iterations without convergence" 错误。

**原因：** Proteus 的 SPICE 求解器将原理图上所有元件作为整体求解，共用 GND/VCC 网络时，数字器件的开关动作与模拟器件的连续求解产生冲突。

**解决方案：**
1. 将 PWM 驱动电路和转速检测电路分别放在不同的 Proteus 原理图页
2. 或使用 "Exclude from simulation" 功能，仿真时只激活一个电路
3. 或调整仿真收敛参数（RELTOL=0.01，ITL4=200）

### 5.2 PC817A 光耦上拉电阻选择

**问题：** 使用 10kΩ 上拉电阻时，MOSFET 栅极波形上升沿为指数曲线，MOSFET 无法完全导通，电机不转。

**原因：** 10kΩ × 1600pF（栅极电容）= 16μs 时间常数，上升沿太慢。

**解决：** 将上拉电阻改为 **470Ω**，时间常数降至 0.75μs，波形接近方波。

### 5.3 光耦反相逻辑

**问题：** PC817A 输出与输入反相，导致占空比越大电机越慢。

**说明：** 这是光耦固有特性，不影响功能，只需在操作时注意逻辑关系。

## 七、总结

本设计完成了直流电机转速检测与脉宽调速器的全部功能：

1. **PWM 调速** — 采用 NE555 + 二极管引导电路，实现频率和占空比独立可调，通过 PC817A 光耦隔离驱动 IRL540 MOSFET 控制电机转速。

2. **转速检测** — 利用电刷噪声法，通过 PC817 光耦采集电机换向噪声，经 CD40106 施密特触发器整形为数字脉冲。

3. **转速显示** — 采用 CD40110 计数/译码/驱动一体化芯片级联，配合差分时序电路实现先锁存后清零的测量方式，2 位数码管显示转速。

4. **关键创新点：**
   - 二极管引导充放电路径，实现 NE555 频率与占空比独立调节
   - 470Ω 上拉电阻解决光耦驱动 MOSFET 的 RC 延迟问题
   - 光耦隔离保证控制电路与功率电路的电气隔离
