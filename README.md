## 系统框图

**主回路 (PWM调速)**

```mermaid
graph TD
    A["12V直流电源"] --> B["LM7805 稳压"]
    B --> C["NE555 多谐振荡器 (f=3~5kHz, D=30~70%)"]
    C --> D["前置放大"]
    D --> E["IRF540N MOSFET"]
    E --> F["直流电机 12V/0.5A + 遮光码盘"]
    E --> G["1N4007 续流二极管"]
```

**测速回路 (转速检测与显示)**

```mermaid
graph TD
    A["槽型光电传感器"] --> B["74HC14 施密特触发器"]
    B --> C["74HC08 与门 (闸门)"]
    C --> D["CD4518 BCD计数器"]
    D --> E["CD4511 BCD-7段译码器"]
    E --> F["2位共阴极数码管 (00~99 r/s)"]
```

**控制回路 (时序基准)**

```mermaid
graph TD
    A["32.768kHz 晶振"] --> B["CD4060 14级分频 (输出2Hz)"]
    B --> C["74HC74 D触发器 (输出1Hz)"]
    C --> D["闸门使能 (高电平1秒采样)"]
    C --> E["锁存脉冲 (锁存显示值)"]
    C --> F["清零脉冲 (清零计数器)"]
```

> 📐 三条回路之间的连接关系详见 [system-block-diagram.drawio](docs/system-block-diagram.drawio)（用 [Draw.io](https://app.diagrams.net/) 打开）

## 设计要求

1. **PWM发生器** — 多谐振荡器，频率3~5kHz，占空比30%~70%
2. **功率放大器** — 电机额定电压≤12V，额定电流≤0.5A
3. **转速检测电路** — 光电转换方法，输出脉冲信号
4. **转速显示电路** — 2位数码管显示 (r/s)
