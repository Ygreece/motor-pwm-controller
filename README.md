## 系统框图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                               主回路 (PWM调速)                              │
│  多谐振荡器 ──→ 前置放大器 ──→ 功率放大器 ──→ 直流电机                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                              │
                                              ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                               测速回路                                      │
│  直流电机 ──→ 转速/脉冲变换 ──→ 整形电路 ──→ 闸门电路 ──→ 计数器 ──→ 锁存器 ──→ 显示器  │
└─────────────────────────────────────────────────────────────────────────────┘
                                              ▲
                                              │
┌─────────────────────────────────────────────────────────────────────────────┐
│                               控制回路                                      │
│  时钟电路 ──→ 定时器1 / 定时器2 ──→ 控制：闸门电路、计数器(清零)、锁存器    │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 设计要求

1. **PWM发生器** — 多谐振荡器，频率3~5kHz，占空比30%~70%
2. **功率放大器** — 电机额定电压≤12V，额定电流≤0.5A
3. **转速检测电路** — 光电转换方法，输出脉冲信号
4. **转速显示电路** — 2位数码管显示 (r/s)
