# 详细设计方案（混合方案）

> **最后更新：2026.6.21** — 仿真工具已从 Multisim 切换到 Proteus

## 一、方案概述

本设计采用**混合架构**，结合两套成熟方案的优点：

- **调速部分**：NE555 + 二极管导引电路产生 PWM，经 PC817A 光耦隔离后驱动 IRL540 MOSFET
- **测速显示部分**：CD40110 + CD40106 架构（参考往届优秀设计）

**核心设计原则：** 控制电路与电机驱动电路**必须光耦隔离**（老师强调）。

### 信号流总结

```
[调速] NE555 PWM → 330Ω → PC817A光耦 → 470Ω上拉 → IRL540 Gate → 电机
                                                    ↑
[测速] 电机电刷噪声 → PC817 → CD40106整形 → CD40110计数 → 数码管显示
                                     ↑
[控制] CD40106振荡(1s) → 差分电路 → 锁存 → 清零
```

## 三、单元电路设计

### 模块1：PWM发生器（NE555 + 二极管导引）

**功能：** 产生频率和占空比可调的 PWM 方波。

**电路拓扑（Proteus 验证可行）：**

```
                    +12V
                     │
                     R2 (100kΩ 电位器 - 频率调节)
                     │
                     ├──── D (1N4148，正极朝节点A)
                     │
                     │     ┌───────────┐
                     │     │   VCC     │
                     │     │           │
                     │     │   RST     ├── +12V
                     │     │           │
                     │     │   OUT     ├── 330Ω → PC817A Pin1
                     │     │           │
                     │     │   CV      ├── C2(10nF) → GND
                     │     │           │
                     │     │   THR     ├──┐
                     │     │           │  │
                     │     │   TRG     ├──┤
                     │     │           │  │
                     │     │   DIS     ├──┤── R3 (100kΩ 电位器 - 占空比调节)
                     │     │           │  │
                     │     │   GND     │  │
                     │     └─────┬─────┘  │
                     │           │        │
                     │          GND       │
                     │                   │
                     │              C1 (100nF)
                     │                   │
                     │                  GND
```

**充电路径（Pin3=HIGH，Pin7开路）：**
```
+12V → R2(100kΩ) → D(1N4148，正向导通) → C1(100nF) → GND
```

**放电路径（Pin3=LOW，Pin7接地）：**
```
C1(100nF) → R3(100kΩ) → Pin7(内部接地) → GND
            D(1N4148，反向截止) 隔断 R2
```

**计算公式：**
- 频率: `f = 1.44 / ((R2 + R3) × C1)`
- 占空比: `D = R2 / (R2 + R3) × 100%`

**参数验证（R2=R3=100kΩ, C1=100nF）：**

| 条件 | R2 | R3 | f | D |
|------|----|----|---|---|
| 中点 | 50kΩ | 50kΩ | 144Hz | 50% |
| R2最小 | 0Ω | 100kΩ | 144Hz | 0% |
| R2最大 | 100kΩ | 0Ω | 144Hz | 100% |

> 注：实际通过调节电位器，可覆盖 3~5kHz 频率范围和 30%~70% 占空比范围。

### 模块2：光耦隔离 + 功率驱动电路

**功能：** 将 NE555 的 PWM 信号隔离传输到 12V 电机驱动侧，驱动 MOSFET 开关电机。

**电路设计（Proteus 验证可行）：**

```
NE555 Pin3 ── 330Ω ── PC817A Pin1 (LED 阳极)
                       PC817A Pin2 (LED 阴极) ── GND
                       PC817A Pin4 (光敏管集电极) ──┬── 470Ω ── +12V
                                                   │
                                                   └── IRL540 Gate
                       PC817A Pin3 (光敏管发射极) ── GND

+12V ── 电机 ── IRL540 Drain
                 IRL540 Source ── GND
```

**工作原理：**

| NE555 输出 | PC817A LED | Pin4 集电极 | IRL540 Gate | MOSFET | 电机 |
|------------|-----------|------------|------------|--------|------|
| HIGH (12V) | 亮 | 拉低 (≈0V) | 0V | 关断 | 停 |
| LOW (0V) | 灭 | 上拉 (12V) | 12V | 导通 | 转 |

**⚠️ 注意：光耦反相！** 占空比越小，电机越快。

**关键参数：**

| 元件 | 值 | 选择理由 |
|------|------|---------|
| R_LED | 330Ω | LED 电流 ≈ 32.7mA，PC817A 最大 50mA |
| R_pull | **470Ω** | 10kΩ 时 RC 时间常数太大（16μs），470Ω 时仅 0.75μs |

> **重要发现：** 上拉电阻必须用 470Ω，不能用 10kΩ。10kΩ × MOSFET 栅极电容(1600pF) = 16μs，上升沿为指数曲线，MOSFET 无法完全导通。

### 模块3：转速检测电路

**功能：** 检测电机转速，输出数字脉冲信号。

**实际硬件方案：光电门检测**

采用槽型光电传感器 + 遮光码盘，电机每转一圈产生一个脉冲：

```
电机轴 ── 遮光码盘 ── 槽型光电传感器 ── CD40106 (施密特整形) ── 数字脉冲
                         │
                    VCC + 限流电阻
```

- 槽型光电传感器模块（LM393 比较器输出，5V CMOS 电平）
- 遮光码盘安装在电机轴上，每转一圈遮光一次
- 脉冲频率 = 转速 (r/s)

**Proteus 仿真替代方案：电刷噪声法**

Proteus 中无法模拟光电门，改用电刷噪声法（参考 CSDN 设计）：

```
电机电刷噪声 ── PC817 (检测光耦) ── CD40106 (施密特整形) ── 数字脉冲
                                       │
                                  偏置/灵敏度调节
                                  W2(1000kΩ) + R29 + R35
```

> **注意：** 电刷噪声法仅用于仿真验证，实际硬件使用光电门方案。

### 模块4：时基振荡与差分时序

**功能：** 产生 1 秒周期时基，控制"先锁存、后清零"的测量时序。

**CD40106 振荡电路：**
```
CD40106 门D ──┬── W2(50kΩ) + R29(25kΩ) ──┬── 输出
              │                            │
              └── C1(22μF) ── GND          │
                                           │
                    调节 W2 使周期 ≈ 1 秒    │
```

**差分电路（先锁存 LE，后清零 RST）：**
```
振荡输出 ──┬── C_diff(0.1μF) ── R_diff(10kΩ) ──→ CD40110 Pin4 (LE)
           │
           └── RC 延时 ──→ CD40110 Pin5 (RST)
```

**⚠️ 时序关键：先锁存(LE)、后清零(RST)，顺序不可反！**

---

### 模块5：计数与显示电路（CD40110）

**功能：** 计数转速脉冲，译码驱动 2 位数码管显示 r/s。

**CD40110 级联：**
```
CD40110 (U1, 个位)
  Pin9  (CLKUP) ──── 整形后的转速脉冲（来自 CD40106）
  Pin7  (CLKDN) ──── GND（不用减计数）
  Pin4  (LE)     ──── 锁存脉冲（来自差分电路）
  Pin5  (RST)    ──── 清零脉冲（来自延时电路）
  Pin6  (TE)     ──── GND（使能计数）
  Pin10 (CARRY)  ───→ U2 Pin9 (CLKUP)
  a~g             ───→ 个位数码管（共阴极）

CD40110 (U2, 十位)
  Pin9  (CLKUP) ──── U1 Pin10 (CARRY 进位)
  Pin7  (CLKDN) ──── GND
  Pin4  (LE)     ──── 锁存脉冲（与 U1 并联）
  Pin5  (RST)    ──── 清零脉冲（与 U1 并联）
  Pin6  (TE)     ──── GND
  Pin10 (CARRY)  ──── 空
  a~g             ───→ 十位数码管（共阴极）
```

```
                                        ↓

```

## 六、关键问题与解决方案

### 6.1 Proteus 混合仿真收敛问题

**问题：** NE555（模拟器件）与 CD40106/CD40110（数字器件）放在同一原理图时，即使没有物理连接，也会报收敛错误。

**解决方案：**
1. 分开仿真（推荐）
2. 调整 SPICE 收敛参数（RELTOL=0.01，ITL4=200）

### 6.2 PC817A 上拉电阻选择

**问题：** 10kΩ 上拉电阻导致 MOSFET 栅极波形为指数曲线。

**解决：** 改用 470Ω，RC 时间常数从 16μs 降至 0.75μs。

### 6.3 光耦反相逻辑

**说明：** PC817A 固有反相特性，占空比越小电机越快，不影响功能。
