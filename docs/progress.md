### 关键问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| Multisim 电位器不工作 | 模型 bug | 切换到 Proteus |
| Proteus NE555 收敛错误 | 混合仿真 SPICE 冲突 | 分开仿真 / 调参 |
| MOSFET 不完全导通 | 上拉 10kΩ RC 太大 | 改用 470Ω |
| 光耦反相 | PC817A 固有特性 | 接受反相逻辑 |
| 两部分电路不能共存 | Proteus 全局求解 | 分开原理图页 |
